module SimSchedulerBasicP

The event queue is the core of the mote side of TOSSIM. It is a wrapper around the underlying heap. Unlike the 1.x version, it is not re-entrant: merging the Python console and TOSSIM means that functionality like packet injection/reception from external tools is on the Python side.
