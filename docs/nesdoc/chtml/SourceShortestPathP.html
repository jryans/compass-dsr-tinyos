Component: SourceShortestPathP

generic module SourceShortestPathP(graph_vertex_id_t maxVertices, typedef graph_edge_weight_t)

Uses Djikstra's algorithm with a priority queue holding fringe vertices to find the single source shortest path. All weights are assumed to be non-negative.
