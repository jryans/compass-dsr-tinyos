Component: tos.chips.atm128.timer.sim.HplAtm128Timer0AsyncC

configuration HplAtm128Timer0AsyncC

The TOSSIM implementation of the Atm128 Timer0. It is built from a timer-specific counter component and a generic compare component. The counter component has an additional simulation-only interface to let the compare component know when its state has changed (e.g., TCNTX was set).
