Component: PacketTransceiverC

generic configuration PacketTransceiverC(am_id_t lowerId)

Multihop protocols instantiate this component to connect the packet engine to the AM layer for a given link layer AM type. The Packet interface used by this component should be provided by the protocol. Similarly, the SubPacket interface provided by this component should be used by the protocol to complete the chain of Packet interfaces.
