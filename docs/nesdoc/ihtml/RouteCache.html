Interface: RouteCache

interface RouteCache

Access some kind of route cache or table that stores information about paths or links in the network and looks for routes when requested.

Author:
Ryan Stinnett

Commands
command void addPath(route_cache_node_id_t *nodes, route_cache_link_id_t numLinks, bool reverse) Adds a path to the route cache with a given number of links along the path and optionally adds the reverse path as well.
command void checkRoute(route_cache_node_id_t src, route_cache_node_id_t dest) Requests an initial check for a route from node src to node dest.
command void removeLink(route_cache_node_id_t src, route_cache_node_id_t dest, bool reverse) Removes any information stored for a given link from the route cache.
command void trackRoute(route_cache_node_id_t src, route_cache_node_id_t dest) Indicates the need for a route from node src to node dest.

Events
event void routeUpdate(route_cache_node_id_t src, route_cache_node_id_t dest, route_cache_link_id_t numLinks) Provides the newest information available pertaining to a requested route.

Commands - Details

addPath

command void addPath(route_cache_node_id_t *nodes, route_cache_link_id_t numLinks, bool reverse)

Adds a path to the route cache with a given number of links along the path and optionally adds the reverse path as well.

Parameters:
nodes - An array of nodes that make up the path to add.
numLinks - The number of links along the path.
reverse - If TRUE, the reverse path is also added.

checkRoute

command void checkRoute(route_cache_node_id_t src, route_cache_node_id_t dest)

Requests an initial check for a route from node src to node dest.

Parameters:
src - The source node of the route.
dest - The destination node of the route.

removeLink

command void removeLink(route_cache_node_id_t src, route_cache_node_id_t dest, bool reverse)

Removes any information stored for a given link from the route cache. If the cache stores entire paths, then any path containing the link must at least be shortened enough to remove the given link or else discarded entirely.

Parameters:
src - The source node in the link to remove.
dest - The destination node in the link to remove.
reverse - If TRUE, the reverse path is also removed.

trackRoute

command void trackRoute(route_cache_node_id_t src, route_cache_node_id_t dest)

Indicates the need for a route from node src to node dest. This is tracked by the route cache until the need is met.

Parameters:
src - The source node of the route.
dest - The destination node of the route.

Events - Details

routeUpdate

event void routeUpdate(route_cache_node_id_t src, route_cache_node_id_t dest, route_cache_link_id_t numLinks)

Provides the newest information available pertaining to a requested route. Whether or not a route is available when the request is made, a routeUpdate will be signaled once the current availability is known. If the route is not initially available, a subsequent routeUpdate will be signaled if a route is later found and the route was requested via the trackRoute command. If a route was found, numLinks is the number of links along the route. If a route was not found, numLinks is 0. The route itself is accessible via the RouteNodeList interface. Note that RouteNodeList traverses the route in reverse, returning the previous or source node for a given node in the route.

Parameters:
src - The source node of the route.
dest - The destination node of the route.
numLinks - Number of links along the route if one was found, 0 if no route was found.
