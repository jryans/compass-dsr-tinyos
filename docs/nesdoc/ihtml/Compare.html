Interface: Compare

interface Compare<typedef compare_t>

Generalized comparison interface between two items. Many uses are possible here, since the inputs could be array indices, as just one example.

Author:
Ryan Stinnett

Commands
command int16_t test(compare_t a, compare_t b) Returns a negative integer, zero, or a positive integer if the first argument is less than, equal to, or greater than the second.
