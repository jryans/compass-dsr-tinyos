Commands - Details

add

command error_t add(message_t *pkt, uint8_t len)

Adds an instance of a type to the TLV block and sets the initial length of the instance. This operation moves data around and should always be performed before storing a value of the given type. The type instance will be added to the block as long as there are at least len + 2 bytes available in the buffer holding the TLV block.

Parameters:
pkt - Pointer to the packet.
len - Length to set for this TLV type.
Returns:
SUCCESS if enough space was available and the length was set, EALREADY if an instance already exists, ESIZE if there wasn't enough space.

length

command uint8_t length(message_t *pkt)

Returns the length of this type instance within the TLV block. If the type instance is not found in the block, then 0 is returned.

Parameters:
pkt - Pointer to the packet.
Returns:
Length of this TLV type.

remove

command error_t remove(message_t *pkt)

Removes this type instance from the TLV block.

Parameters:
pkt - Pointer to the packet.
Returns:
SUCCESS if the type instance was found and removed, EINVAL otherwise.

setLength

command error_t setLength(message_t *pkt, uint8_t len)

Sets the length of a type instance within the TLV block. This operation moves data around and should always be performed before storing a value with a different length than the one currently stored. If the type instance is not found in the block or there is not enough room for the new length, returns ESIZE.

Parameters:
pkt - Pointer to the packet.
len - Length to set for this TLV type.
Returns:
SUCCESS if the type instance was found, enough space was available, and the length was set, ESIZE if there wasn't enough space, EINVAL if the instance wasn't found.

value

command void *value(message_t *pkt)

Returns the value in this type instance within the TLV block. If the type instance is not found in the block, then NULL is returned.

Parameters:
pkt - Pointer to the packet.
Returns:
Pointer to the value in this TLV type.
