Interface: TlvBlockPacket

interface TlvBlockPacket

Manipulate a packet with a TLV block independent of the types it contains.

Author:
Ryan Stinnett

Commands
command error_t add(message_t *pkt) Adds a TLV block at the beginning of a given packet.
command uint8_t length(message_t *pkt) Returns the length of the entire TLV block, including the block length value itself.
command void remove(message_t *pkt) Removes a TLV block from the beginning of a given packet.

Commands - Details

add

command error_t add(message_t *pkt)

Adds a TLV block at the beginning of a given packet. Other data is moved to accomidate it. This must be called on any new TLV block before using any other TLV commands.

Parameters:
pkt - Pointer to the packet.
Returns:
SUCCESS if the add completed, ESIZE if there wasn't enough space.

length

command uint8_t length(message_t *pkt)

Returns the length of the entire TLV block, including the block length value itself.

Parameters:
pkt - Pointer to the packet.
Returns:
Length of the TLV block.

remove

command void remove(message_t *pkt)

Removes a TLV block from the beginning of a given packet. Other data is moved to fill the gap.

Parameters:
pkt - Pointer to the packet.
