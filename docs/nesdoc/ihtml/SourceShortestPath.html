Interface: SourceShortestPath

interface SourceShortestPath

Access details on the shortest path between vertices of a graph.

Author:
Ryan Stinnett

Commands
command bool buildingTree() Checks if the tree is currently being built.
command error_t buildTree(graph_vertex_id_t src) Build a shortest path tree from vertex src.
command error_t buildTreeTo(graph_vertex_id_t src, graph_vertex_id_t dest) Build the shortest path from vertex src to vertex dest.
command graph_edge_id_t edgeCount(graph_vertex_id_t dest) Get the number of edges to the destination vertex.
command bool rebuildNeeded(graph_vertex_id_t src) Checks if the tree should be rebuilt either because the source has changed or because the underlying graph structure has changed.

Events
event void builtTree(graph_vertex_id_t src) The shortest path tree from vertex src is complete.
event void builtTreeTo(graph_vertex_id_t src, graph_vertex_id_t dest, error_t pathResult) The shortest path from vertex src to vertex dest is complete.

Commands - Details

buildingTree

command bool buildingTree()

Checks if the tree is currently being built.

Returns:
Whether the tree is being built.

buildTree

command error_t buildTree(graph_vertex_id_t src)

Build a shortest path tree from vertex src.

Parameters:
src - The source vertex.
Returns:
If the vertex is too large then ESIZE or if a tree is currently being built then EBUSY, else SUCCESS.

buildTreeTo

command error_t buildTreeTo(graph_vertex_id_t src, graph_vertex_id_t dest)

Build the shortest path from vertex src to vertex dest.

Parameters:
src - The source vertex.
dest - The destination vertex.
Returns:
If the vertex is too large then ESIZE or if a tree is currently being built then EBUSY, else SUCCESS.

edgeCount

command graph_edge_id_t edgeCount(graph_vertex_id_t dest)

Get the number of edges to the destination vertex. For example, if the shortest path from A to C was A -> D -> C, edgeCount() would return 2.

Parameters:
dest - The destination vertex.
Returns:
The number of edges. If there is no path to the given destination, or if an input vertex is too large, then the count is 0.

rebuildNeeded

command bool rebuildNeeded(graph_vertex_id_t src)

Checks if the tree should be rebuilt either because the source has changed or because the underlying graph structure has changed.

Parameters:
src - The source vertex.
Returns:
Whether a rebuild is needed.
