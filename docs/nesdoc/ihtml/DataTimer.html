Interface: DataTimer

interface DataTimer<typedef data_t>

Commands
command error_t start(data_t data, data_timer_time_t delay) Starts the non-periodic timer for a given data item.
command error_t stop(data_t data) Stops the timer for a given data item.

Events
event void expired(data_t data) The timer for a given data item has expired.

Commands - Details

start

command error_t start(data_t data, data_timer_time_t delay)

Starts the non-periodic timer for a given data item. If the same item is started multiple times, it will be tracked multiple times.

Parameters:
data - The data item to start timing.
delay - Time units before the data expires.
Returns:
SUCCESS, or FAIL if already full.

stop

command error_t stop(data_t data)

Stops the timer for a given data item. If the item was started multiple times, this removes the instance that was closest to expiring.

Parameters:
data - The data item to stop timing.
Returns:
SUCCESS, or EINVAL if the item was not found.

Events - Details

expired

event void expired(data_t data)

The timer for a given data item has expired.

Parameters:
data - The data item which expired.
