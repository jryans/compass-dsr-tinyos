Interface: DataCache

interface DataCache<typedef item_t>

A data cache. While this is quite similar to the Cache interface, here a pointer to the element in the cache matching the given input item is returned. Additional features, such as remove and eviction, are also available.

Author:
Ryan Stinnett

Commands
command void flush() Flush the cache of all entries.
command item_t *insert(item_t *item) Inserts an item in the cache, evicting if necessary.
command item_t *lookup(item_t *item) Returns a pointer to the item in the cache by matching various keys or NULL if it was not found.
command error_t remove(item_t *item) Removes an item from the cache.
command uint8_t size() Returns the number of items currently stored in the cache.

Events
event void evicted(item_t *item) Notification that the given item is about to be evicted due to an insert which was requested when the cache was already full.

Commands - Details

flush

command void flush()

Flush the cache of all entries.

insert

command item_t *insert(item_t *item)

Inserts an item in the cache, evicting if necessary. An atomic lookup after insert should return true.

Parameters:
item - Pointer to the data item to insert into the cache.
Returns:
A pointer to the item in the cache.

lookup

command item_t *lookup(item_t *item)

Returns a pointer to the item in the cache by matching various keys or NULL if it was not found.

Parameters:
item - Pointer to the data item to find in the cache.
Returns:
A pointer to the item in the cache, or NULL if not found.

remove

command error_t remove(item_t *item)

Removes an item from the cache.

Parameters:
item - Pointer to the data item to remove from the cache.
Returns:
SUCCESS if found and removed, EINVAL if not found.

size

command uint8_t size()

Returns the number of items currently stored in the cache.

Returns:
The number of items.

Events - Details

evicted

event void evicted(item_t *item)

Notification that the given item is about to be evicted due to an insert which was requested when the cache was already full. Note that this is not signaled on calls of remove. The pointer passed here should not saved, as it will not longer be vaild once this event returns.

Parameters:
item - Pointer to the data item being evicted.
