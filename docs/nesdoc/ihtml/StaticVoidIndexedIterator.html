Interface: StaticVoidIndexedIterator

interface StaticVoidIndexedIterator<typedef iterator_item_t>

Iteration operations using indices. This interface is the static or stateless version which requires a pointer to the structure containing the data to be passed in along with the index.

Author:
Ryan Stinnett
