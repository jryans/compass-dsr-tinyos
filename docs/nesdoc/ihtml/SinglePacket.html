Interface: SinglePacket

interface SinglePacket

Access and set the properties of a multihop packet with a single node address based destination.

Author:
Ryan Stinnett

Commands
command am_addr_t destination(message_t *msg) Returns the address of the destination of the multihop packet.
command bool isForMe(message_t *msg) As with the AM layer, returns whether this node is the destination for the packet, but can also include special processing for broadcast addresses or similar values.
command void setDestination(message_t *msg, am_addr_t dest) Sets the address of the destination of the multihop packet.
command void setSource(message_t *msg, am_addr_t src) Sets the address of the source of the multihop packet.
command am_addr_t source(message_t *msg) Returns the address of the source of the multihop packet.

setDestination

command void setDestination(message_t *msg, am_addr_t dest)

Sets the address of the destination of the multihop packet. As with the AM layer, this is set by the multihop layer, and should not need to be used when sending a packet. However, buffers or other storage components may wish to use it to store the address for later retrieval.

Parameters:
msg - Pointer to the packet.
dest - The destination node of the multihop packet.
