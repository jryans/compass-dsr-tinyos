Interface: Equal

interface Equal<typedef equal_t>

Generalized equality interface between two items. Many uses are possible here, since the inputs could be array indices, as just one example.

Author:
Ryan Stinnett

Commands
command bool test(equal_t a, equal_t b) Checks whether two items are equal.

Commands - Details

test

command bool test(equal_t a, equal_t b)

Checks whether two items are equal.

Parameters:
a - First item to test.
b - Second item to test.
Returns:
TRUE if they are equal, FALSE if they are not.
