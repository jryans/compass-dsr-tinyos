Interface: PacketEngineClient

interface PacketEngineClient

Stores the upper AM type and default lower AM type for a packet engine client. This information aids in moving between the two kinds of AM types.

Author:
Ryan Stinnett

Commands
command am_id_t defaultLowerType() Returns the default lower AM type packets sent by this client should initially be set to.
command am_id_t upperType() Returns the upper AM type used by this client above the the packet engine layer.
