Interface: GraphEdgeWeight

interface GraphEdgeWeight<typedef weight_t>

Access information about the weight of an edge of the graph.

Author:
Ryan Stinnett

Commands
command bool canChangeWeights() Checks if the graph weights can be changed.
command void setWeight(graph_edge_id_t m, weight_t w) Set the weight of the edge.
command weight_t weight(graph_edge_id_t m) Get the weight of the edge.

Events
event void weightChanged(graph_edge_id_t m) The weight of an edge in the graph changed.

Commands - Details

canChangeWeights

command bool canChangeWeights()

Checks if the graph weights can be changed. This value should not change at run-time.

Returns:
TRUE if the graph weights can change, FALSE if they cannot.

setWeight

command void setWeight(graph_edge_id_t m, weight_t w)

Set the weight of the edge. If the graph is actually unweighted, no action is taken.

Parameters:
m - The key for the edge.
w - The updated weight.

weight

command weight_t weight(graph_edge_id_t m)

Get the weight of the edge. If the graph is actually unweighted, a unit weight is always returned.

Parameters:
m - The key for the edge.
Returns:
The weight of the edge.

Events - Details

weightChanged

event void weightChanged(graph_edge_id_t m)

The weight of an edge in the graph changed.

Parameters:
m - The key for the edge that had its weight changed.
