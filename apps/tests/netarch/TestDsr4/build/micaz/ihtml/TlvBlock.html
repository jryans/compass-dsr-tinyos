Interface: TlvBlock

interface TlvBlock

Manipulate a TLV block independent of the types it contains.

Author:
Ryan Stinnett

Commands
command error_t add(void *block, uint8_t trailLen, uint8_t maxLen) Adds a TLV block at the beginning of a given data buffer.
command uint8_t length(void *block) Returns the length of the entire TLV block, including the block length value itself.

Commands - Details

add

command error_t add(void *block, uint8_t trailLen, uint8_t maxLen)

Adds a TLV block at the beginning of a given data buffer. Other data is moved to accomidate it. This must be called on any new TLV block before using any other TLV commands.

Parameters:
block - Pointer to the TLV block.
trailLen - Current length of trailing data.
maxLen - Maximum possible length of TLV block and trailing data.
Returns:
SUCCESS if the add completed, ESIZE if there wasn't enough space.

length

command uint8_t length(void *block)

Returns the length of the entire TLV block, including the block length value itself.

Parameters:
block - Pointer to the TLV block.
Returns:
Length of the TLV block.

remove

command void remove(void *block, uint8_t trailLen)

Removes a TLV block from the beginning of a given data buffer. Other data is moved to fill the gap.

Parameters:
block - Pointer to the TLV block.
trailLen - Current length of trailing data.
